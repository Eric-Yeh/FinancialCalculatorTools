財務規劃工具箱使用說明書

個人財務總管 (cashflow.html)

用途：每月例行記帳、資產盤點。

重點：務必勾選資產/負債前方的核取方塊，將配息或利息支出帶入現金流，才能看到真實的月收支結餘。

雙向貸款計算機 (loan.html)

用途：買房、買車、貸款前評估。

重點：右側的「利率反推」功能可用於拆解民間借貸或分期付款的真實成本，避免落入高利陷阱。

定期定額/複利計算機 (compound.html)

用途：設定長期目標（如退休、買房頭期款）。

重點：觀察長期持有的複利效應，建立長期投資的信心。右側有一個通膨計算機，用以確保資產增長速度高於通膨率 (2.5%~3%)。
